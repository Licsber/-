#  五彩斑斓的黑

## 项目简介

把pdf文件转化为五彩斑斓的pdf！  
这样看黑白论文的时候就不无聊了（

## 效果图

![](https://github.com/Licsber/Colorful-Black/blob/master/example.jpg)

## 环境依赖

```
conda install flask imagemagick wand ghostscript opencv -c conda-forge
yum install mesa-libGL.x86_64
```

## 运行

```
python server.py
```
